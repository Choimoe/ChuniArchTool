Rin服存档转落雪成绩

上传你的 RinNet JSON 档案，系统将自动转换为落雪格式的 CSV 成绩文件并提供下载。

重要提示

落雪的上传成绩操作会清除你账号内现有的所有成绩。请务必在上传前先导出并备份原始数据，同时不要删除任何现有存档，避免造成无法恢复的损失。

建议：先下载原数据 → 使用本工具转换 → 合并/上传到落雪。

LxNet 上传成绩页面： https://maimai.lxns.net/user/scores?tab=backup （用于上传转换后的 CSV 文件）
RinNet 导出存档页面： https://portal.naominet.live/chuni/v2/setting （网页下方有“导出玩家存档”按钮，导出 JSON 文件）

前往 CSV 合并工具

CSV 文件合并

上传两个 CSV 文件。系统将使用第一个文件的表头，并将第二个文件的数据追加到第一个文件后。请确保两个文件的表头（列）完全一致。
